# ♞ Knight Trainer

> *"Fischer is Fischer, but a knight is a knight!"* — Mikhail Tal

**800,000+ knight-focused chess puzzles** compiled from Lichess' Tactics Trainer. From gentle foals to ferocious stallions.

---

## What It Is

Knight Trainer is a dedicated training tool for one of chess's most deceptively dangerous pieces.

---

## Features

| Feature | Description |
|--------|-------------|
| **800,000+ Puzzles** | Sourced from Lichess Tactics Trainer, filtered for knight motifs |
| **Themed Collections** | Forks, smothered mates, outpost creation, queen-and-knight coordination, blockades etc |
| **3 Difficulty Levels** | Calibrated for beginners to seasoned club players |
| **Clean, Fast UI** | No clutter, no distractions, no gamification gimmicks — just you and the horse |
| **Pattern Recognition Training** | Rewire your brain to see the board through the knight's eyes |

---

## Puzzle Themes

- **Forks** — The knight's signature move. Learn to spot them. Learn to deliver them.
- **Smothered Mates** — The most elegant checkmate in chess. A lone horse choking a king in the embrace of his own pieces.
- **Outposts** — The octopus knight. Establish unassailable dominance on key squares.
- **Queen & Knight Coordination** — The classic attacking duo.
- **Blockades** — Paralyze enemy pawns and cramp your opponent's position.

---

## Why Train the Knight?

> *"The knight's movement activates divergent thinking — the ability to see possibilities that are not immediately obvious."*

Our brains are wired for linear thinking. The knight's L-shaped leap doesn't fit. Training with it rewires how you process the entire board. In blitz and bullet, where time pressure scrambles your brain, the knight's value actually increases — it's harder to defend against when you're low on the clock.

```bash

# Navigate to the project

# Install dependencies
npm install

# Start the development server
npm run dev
```
